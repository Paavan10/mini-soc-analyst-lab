# Mini SOC Analyst Lab – SSH Brute Force Detection

## Project Overview
This project simulates a real-world Security Operations Center (SOC) investigation by detecting an SSH brute-force attack using Kali Linux.

## Objectives
- Perform reconnaissance using Nmap
- Simulate brute-force attack using Hydra
- Analyze authentication logs
- Identify Indicators of Compromise (IOC)
- Create Incident Response Report

## Tools Used
- Kali Linux
- Nmap
- Hydra
- Linux journalctl logs

## Attack Simulation
An SSH brute-force attack was simulated against localhost to generate real security logs.

## Indicators of Compromise
- Multiple failed login attempts
- Repeated authentication failures
- SSH service targeted

## Mitigation Steps
- Disable password authentication
- Use SSH keys
- Implement Fail2Ban
